Manual steps to test each individual contract and verify if the behavior is as expected

STEP 0: Deploy and set contract addresses

1 - Deploy each smart contract (Decision, Attributes, DataAccess and Policies) and save the addresses of each
2 - DataAccess: run setAttributesContractAddr and setDecisionContractAddr
3 - Decision run setAttributesContractAddr, setPoliciesContractAddr and setDataAccessContractAddr
4 - Policies run setAttributesContractAddr
